# How to Run
```bash
    $ chmod +x run.sh
    $ ./run.sh
```

# Dependencies
```
    1. lex
    2. yacc
```

# Example
```
    $ ./run.sh 
    a = (2^3)*5+6-(4^2)/6;      //user input
    print a;                    //user input
    Printing 43.333333          //OUTPUT
    print a+1;                  //user input
    Printing 44.333333          //OUTPUT
                    
```
